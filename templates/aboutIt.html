What is Post.it?

Post.it allows users to customize their social medias to have multiple Youtube videos, Pinterest boards, and Twitter feeds on one site at the same time. Users are also able to Tweet and Pin directly from the Post.it site.

Why was Post.It Developed?

There are multiple Social Medias such as Twitter, Facebook, etc. however, when users want to view social media content, they must switch between each individual site. Post.it prioritizes user experience and allows all content to be embedded in one place at the same time.

How Does Post.it Work?

Users are able to customize the Pinterest boards, Youtube videos, and Twitter feeds by simply entering the URL of the prospective Social Medias once. The social media is saved and embedded to their profile. Users can embed an infinite number of Social Media links.

Can anyone else see what I have on my Poster?

Post.it only allows each user to access their own embedded social media. However, if a user tweets or pins it will link to that profile.
